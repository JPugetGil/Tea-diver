## Lancement du projet

Installation des dépendances :
```
yarn install
```

ou

```
npm install
```

Build le projet :
```
yarn dev
```

ou

```
npm run dev
```

Lancement du serveur :
```
yarn start
```

ou

```
npm run start
```

Lancement de l'écoute de L'arduino :
```
yarn arduino
```

ou

```
npm run arduino
```
